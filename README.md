# 🧩 PRODIGY_SD_04 – Sudoku Solver

A modern Sudoku Solver built using Python and CustomTkinter. The application uses the Backtracking Algorithm to automatically solve Sudoku puzzles with a clean and interactive graphical user interface.

---

## 🚀 Features

- 🧠 Automatic Sudoku Solving
- 🎲 Load Sample Puzzle
- 🗑 Clear Board
- 🌙 Modern Dark-Themed GUI
- ⚡ Fast Backtracking Algorithm
- 📋 Interactive 9x9 Sudoku Grid

---

## 🛠 Technologies Used

- Python
- CustomTkinter
- Backtracking Algorithm

---

## ▶ How to Run

### 1️⃣ Install Required Library

```bash
pip install customtkinter
```

### 2️⃣ Run the Program

```bash
python sudoku_solver.py
```
